# Products App

Marketplace-style iOS application built with Swift and UIKit.
The app fetches products from FakeStore API and displays them in a searchable list.

## Features

- Products list (UITableView)
- Product search
- Product detail screen
- Loading indicator
- Error handling
- Order imitation (alert)

## Tech Stack

- Swift
- UIKit
- URLSession
- async/await
- Codable
- MVC architecture

## API

FakeStore API

## Implementation Details

- Networking layer separation
- JSON parsing
- Asynchronous programming
- UI updates on main thread
- Error handling based on HTTP status codes
